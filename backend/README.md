# Backend (Express)

Express API that ingests timetable images/PDFs, preprocesses them, performs OCR, detects table structures, and parses events into structured JSON.

## Scripts
- `npm start` – run server
- `npm run dev` – run with nodemon
- `npm run lint` – lint source

## Env
- `PORT` (default 4000)
- `OCR_SERVICE_URL` (optional: EasyOCR microservice)
